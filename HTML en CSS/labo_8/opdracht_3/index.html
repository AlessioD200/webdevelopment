Deze staan gecentreerd doordat ze door css via de code "text-align:center;" in het midden zijn geplaatst.

De vertical-align:center heeft geen effect op inline opbjecten.

de vertical-align:middle zou wel een effect moeten hebben maar zou inline moeten worden geprogrammeerd.

Je kan deze door deze regel te gebruiken "line-height: 100px;"

Doormiddel van display: table-cell en dan vertical-allign: middle; kun je meerdere regels verticaal centreren.

Door de specifiteit van de css-regel bv bij dit voorbeeld heb je .area.tall dat wil zeggen dat het laatste die erin staat een hogere specifiteit heeft bij deze .tall.
